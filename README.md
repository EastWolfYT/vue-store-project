# Vue Store Project

A demo e-commerce web application built with Vue, Vite and Express.js.  
The project includes product browsing, filtering, sorting, cart functionality and simple user authentication.

## Features

- Product listing
- Product details page
- Search and filtering
- Sorting
- Pagination
- Shopping cart
- User registration
- User login
- Cookie-based session handling
- Express.js backend API
- JSON-based mock data

## Tech Stack

### Frontend

- Vue.js
- Vite
- Vue Router
- Vuex
- JavaScript
- CSS

### Backend

- Node.js
- Express.js
- JSON files as mock database
